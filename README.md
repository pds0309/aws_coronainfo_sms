# AWS_CoronaInfo_SMS
----------------
Daily South Korea Corona Information Text Notification Service Using AWS for learning aws computing instance.

AWS 를 활용한 일일 코로나 정보 문자메시지 알림 서비스

### 일시
--------------
2020.12.01~2020.12.03
